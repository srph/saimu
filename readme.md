# Uomi
[WIP] A user-friendly personal debt tracker tool

![Preview](preview.png)

## Developing
```
# Setup
cp .env.example .env # Setup env
npm start # Run build and watch for changes
open public/index.html # Open app

# Build for production
npm prod
```
